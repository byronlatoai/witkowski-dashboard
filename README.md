# Witkowski Dashboard

Operations platform prototype for **Witkowski Mechanical** — a FieldEdge replacement.

Single-page reference prototype with a role-based view switcher (**Admin / Office / Field**) that demonstrates the permission model: each role sees only the modules and panels it is allowed to.

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```

## Status

`v0` prototype — reference design. Brand colors and final modules to be defined.

## Modules (current)

- System Overview (KPIs, today's dispatch, recent activity)
- Integration health (admin)
- Data governance (admin)
- Sidebar navigation: Operations + System groups
- Field portal note (technicians run on a separate mobile app)
